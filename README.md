# SK2-sieci-projekt


Kompilowanie serwera: 

```
gcc server.c -o server 
```

Uruchamianie serwera: 

```
./server 
```

Uruchamianie clienta


```
python client.py 
```


Dla testów na tym samym komputerze najlepiej zmienić adress IP na kliencie i serwerze na **127.0.0.1**
